Blender快捷键(Mac)

2022年09月09日

导读

最近在学习blender，记录一些快捷键，持续更新中

操作 MacOS快捷键 备注
创建物体 ⇧ + A
删除 X
全选 A
取消全选 AA
3D游标回到原点 ⇧ + C
浏览物体 按住鼠标中键
平移视角 ⇧ + 按住鼠标中键
移动物体 G
G + X 沿x轴移动
G + Y 沿y轴移动
G + Z 沿z轴移动
旋转物体 R ±N代表缩放大小正负数
R + X 沿x轴旋转±N精准缩放大小
R + Y 沿y轴旋转±N精准缩放大小
R + Z 沿z轴旋转±N精准缩放大小
缩放物体 S ±N代表缩放大小正负数
S + X 沿x轴缩放±N精准缩放大小
S + Y 沿y轴缩放±N精准缩放大小
S + Z 沿z轴缩放±N精准缩放大小
进入编辑模式 Tab
环切 ⌘ + R
移动循环边 ⌘ + R + 左键
循环边居中 ⌘ + R + 右键
快速添加成倍的循环边 ⌘ + R + 滚轮
倒角 ⌘ + B
挤出 E
E + X 沿x轴挤出
E + Y 沿y轴挤出
E + Z 沿z轴挤出
内插面 I
透视模式 ⌥ + Z
法向缩放 ⌥ + S X,Y轴缩放,Z轴不变
选取点 1
选取边 2
选取面 3
挤出 E
反选 ⌘ + I
调出视图选择快捷窗口 ~
1/2/3/4/5/6/7/8/9（小键盘） 切换视图
⌘+1/2/3/4/5/6/7/8/9（小键盘） 反向切换视图
